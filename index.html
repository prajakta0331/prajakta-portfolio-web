Online Ticket Reservation System

A console-based application demonstrating Object-Oriented Principles in C++.

Core OOP Implementation

[Image of object oriented programming class diagram]
  • Classes & Objects: Structured the system into modular classes such as Bus, Ticket, and SystemAdmin to separate concerns.
  • Encapsulation: Secured passenger data and financial details by using private members and public accessor methods.
  • Abstraction: Provided a simplified interface for users to book seats while hiding the complex 2D-array mapping logic.
  • Inheritance (Concept): Applied hierarchy for different types of transport classes to maximize code reusability.

Technical Features

  • File I/O Handling: Integrated fstream library to ensure data persistence, allowing the system to save reservations to local files.
  • Dynamic Management: Real-time seat availability updates using conditional logic and array manipulation.
  • Input Validation: Implemented robust error checking to prevent overbooking or invalid seat selection.
Tech Stack: C++, Standard Template Library (STL), File Systems.
